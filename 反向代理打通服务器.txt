ssh -L 本地端口:服务器内网IP:服务器端口 用户名@服务器公网IP

例如，假设你的服务器内网 IP 是 192.168.1.100，服务器上运行的项目监听在端口 8080，服务器公网 IP 是 106.53.222.72，你想在本地电脑上使用端口 8000 访问该项目，可以使用以下命令：

ssh -L 8000:192.168.1.100:8080 用户名@106.53.222.72

3. 访问本地端口
在本地电脑上打开浏览器，输入以下地址访问项目：

http://localhost:2300


我的电脑to反向穿透：
ssh  -L 2300:localhost:8188 ubuntu@43.134.183.133

ssh  -L 本地电脑端口:localhost:服务器端口 ubuntu@106.53.222.72
